I miss pacman

I already finished my initial setup for Windows. And I have to say that I am not enjoying it as much as I would enjoy setup any other thing. As the title says I MISS PACMAN.

Actually what I miss is any kind of package manager, I hate going to every single webpage of any application I want to install. Fortunatelly there exists Chocolatey and so far it is fine. I used it to install some applications and it works nicely. I would recommend any Windows user to get familiar with this package manager.

Also I installed VirtualBox because on this time I want to try some things. As I said before, I will use Win10 only for a couple of months and then this computer is going to be my brother's. Certainly I cannot wait to have my hands on a 2011's Thinkpad.

Oh and the things I will try on VB are some operative systems in order to choose one to use for a long time. I've been thinking of using Alpine since is GNU free and it would be nice to stop using the GNU interpretation of every single utility. Also I might try Artix with its different init systems in order to choose one of them. Actually as Alpine uses OpenRC as init system, I might only try Artix on the runit and s6 versions. My safety is of course Arch, but I think being away of systemd might even be beneficial in order to learn some new stuff. I think I will also try FreeBSD and OpenBSD because It might be interesting to use the BSD userland. If you know about any other systemd and free distro feel free to email me

Now, this will be a busy week on work and I might not blog often. But on the weekend I am planning on organsing my digital library with Dewey's help and also to install Artix. Have a nice week and stay home.
